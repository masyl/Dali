Dali by masyl

Javascript Templating Engine

Dali is a modern web templating engine. It aims to provides an powerfull syntax, a rich javascript API and many ways for you to extend its capabilities to your own specific needs. You can use it “client-side” on any browser and on the “server-side” with the Node.Js application server.
