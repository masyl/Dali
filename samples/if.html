Conditionnal (if)

The "if" tag allows you to handle conditionnal operations in many ways. There are 3 arguments to this tag:

  • The expression to be evaluated as true or false
  • The value if true
  • The value if false

You can also provide a block of content to display if the expression is true.

Alternate tags:

  • {{if-else someExpression}} : Try to match another condition if precedent one wasnt.
  • {{if-not}} : If no previous condition have been true, fallback on this tag.
