Iterator (each)

To iterate across arrays and object members, you can use the "each" tag.

This tag behaves similarly to a javascript "for" statement, but does more by automatically setting the current item to the "this" object.

Also, a "loop" object is made available to access the current state of the iteration.

Alternate tags:

  • {{each-first}} : Used for the first item.
  • {{each-last}} : Used for the last item.
  • {{each-begin}} : Content to append before all items if not empty.
  • {{each-between}} : Content to append between each item.
  • {{each-end}} : Content to append after all items if not empty.
  • {{each-single}} : Used when the collection has a single item.
  • {{each-empty}} : Used when the collection to iterate is empty.
