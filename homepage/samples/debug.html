Debuging and Logging

The "debug" and "log" utilities are available both as filters and tags. They can give you visibility on the rendering process without having to set complicated breakpoint in a console.

The "log" tag simply show the rendered output of the tag, while the debug tags also exposes the environement data such as item, vars, env.

You an use a web console sur as Firebug to see the logs and debug informantion.
